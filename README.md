# Wizard

Product notes and design guidance for a dashboard builder that lets business users compose layouts with drag-and-drop cards containing charts or grids. See [`docs/dashboard_layout_tools.md`](docs/dashboard_layout_tools.md) for the layout tools, rendering controls, and configuration model.

## Preview the documentation

1. Start a static server from the repo root: `python -m http.server 8000`.
2. Open `http://localhost:8000/docs/preview.html` in a browser to view the rendered Markdown with styling.
3. Click **Reload** in the preview header if you edit the Markdown and want to refresh the rendering.
